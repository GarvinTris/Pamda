Latar Belakang Aplikasi:
Pamda adalah aplikasi edukasi berbasis kuis kosakata bahasa Mandarin untuk melatih terbiasa dengan huruf bahasa Mandarin atau **hanzi** agar memudahkan para user dalam berkomunikasi dengan luar negeri atau bekerja dan membentuk kalimat Mandarin secara lengkap. Aplikasi ini dibuat untuk semua orang, terutama yang awam terhadap bahasa tersebut. Aplikasi ini memiliki sistem penilaian atau Mastery untuk menilai seberapa mahir user dapat menjawab dengan benar Dan algoritma. Selain itu, saat user masuk ke aplikasi, ia akan masuk ke stage yang terdiri dari 15 stage atau lebih, di mana setiap stage diisi sesuai total kosakata yang tersedia di hsk. Ketika gagal menjawab, jawaban akan ditampilkan di layar. Soal tersebut akan diulang kembali hingga berhasil. Ketika user berhasil mencapai Mastery level 100, user akan dilanjutkan ke stage berikut hingga berakhir, lalu akan dialihkan ke hasil kuis dan dialihkan ke home. User akan memiliki progress streak untuk memantau kebiasaan user dalam menggunakan aplikasi. aplikasi dilengkapi berbagai hsk dari 1 - 7 dan setiap hsk memiliki beda vocabulary dan hanzi yang dimuncul ke layar.

Sistem Utama:
- User masuk ke aplikasi
- Memilih kuis hsk yang tersedia
- User akan di bawa ke soal pilgan yang berbentuk Hanzi
- Selesai stage, langsung ke stage berikutnya
- Muncul hasil kuis, progress streak, benar dan salah jawaban
- Kembali ke Home

Sistem Pendukung:
- Progress Streak
- Save Progress
- Stage

Spesifikasi Teknis:
- Java
- Room
- SQLite3 (Django)

Keunggulan memilih Pamda:
- Ringan dan mudah digunakan
- Aplikasi tidak berat
- Bisa Offline
- Dapat melatih ulang secara gratis

Batasan Aplikasi:
- Aplikasi tidak memiliki fitur pendukung tambahan seperti Text-to-Audio untuk melatih pelafalan dan fitur pengulangan materi untuk memperkuat ingatan
- Fitur timeout tidak dimasukkan, berfokus pada kebiasaan hanzi
- Tampilan aplikasi kurang interaktif secara moderen
- Tidak memiliki sistem login / register
- Keamanan belum diintegrasi

Target Output:
Setelah selesai mengerjakan semua stage, user diharapkan memiliki fondasi karakter yang kuat untuk menempuh ujian hsk atau sertifikat hsk resmi serta memahami konteks penggunaan hanzi di dunia kerja. Dan harapan kami adalah bahwa user menggunakan aplikasi tersebut dengan nyaman.
